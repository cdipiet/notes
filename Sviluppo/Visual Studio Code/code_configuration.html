Ctrl+Click - Go to Definition ðŸ”—

  • https://code.visualstudio.com/docs/editor/editingevolved#_go-to-definition
    • you can go to the definition of a symbol by pressing F12
    • You can jump to the definition with Ctrl+Click
  • https://code.visualstudio.com/docs/editor/codebasics#_multicursor-modifier The Go to Definition and Open Link gestures will also respect this setting and adapt such that they do not conflict. For example, when the setting is ctrlCmd, multiple cursors can be added with Ctrl/Cmd+Click, and opening links or going to definition can be invoked with Alt+Click. ==> Se succede questo bisogna invertire l'impostazione.
  • https://stackoverflow.com/questions/52081768/how-i-can-make-ctrl-click-to-go-to-definition-in-visual-studio-code-editor-f

Se la funzione Go to Definiton funziona solo con F12 e non anche con Ctrl+Click
allora il tasto Control Ã¨ associato alla funzione Multi Cursor Modifier, bisogna associargli invece il tasto Alt:

/File/Preferences/Settings/Text Editor/Multi Cursor Modifier: Alt

settings.json:
"editor.multiCursorModifier": "alt"

Grazie a questa impostazione la combinazione Ctrl+Click torna ad invocare la funzione Go to Definiton.

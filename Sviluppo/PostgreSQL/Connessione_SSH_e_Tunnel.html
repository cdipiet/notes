Autenticazione SSH con chiave pubblica üîó

Key based authentication in SSH is called public key authentication.

The ~/.ssh/authorized_keys file in SSH specifies the SSH keys that can be used for logging into the user account for which the file is configured. It is a highly important configuration file, as it configures permanent access using SSH keys.

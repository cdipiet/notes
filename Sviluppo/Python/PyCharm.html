~/bin/pycharm_install.sh

#!/usr/bin/env bash
VERSION=2021.1.3
EXT=tar.gz
FILE=pycharm-community-$VERSION
TARFILE=$FILE.$EXT
URL=https://download.jetbrains.com/python/$TARFILE
DST=/opt/pycharm
cd /tmp
wget $URL
tar xfz $TARFILE
sudo rm -rf $DST
sudo mv $FILE $DST
sudo chown -R $USER:$USER $DST
rm -rf /tmp/$FILE*

Problema: PyCharm/IntelliJIdea segnala che non riesce a monitorare tutti i file del workspace ðŸ”—

sysctl

Soluzione:

  • https://youtrack.jetbrains.com/articles/IDEA-A-2/Inotify-Watches-Limit

Bisogna incrementare il numero di file controllabili da inotify:

Incrementare il numero di file controllabili da inotify

Utile ad esempio per PyCharm ed AndroidStudio

$ systemd-analyze cat-config sysctl.d
$ man sysctl
$ man sysctl.d

$ sudo sysctl -a|grep fs.inotify.max_user_watches
$ sudo bash -c "cat > /etc/sysctl.d/max_user_watches.conf" << 'EOF'
fs.inotify.max_user_watches = 524288
EOF
sudo sysctl -p --system # ricaricare la configurazione

And don't forget to restart your IDE.

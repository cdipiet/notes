    • pyside6-project, to build Qt Designer forms (.ui files), resource files (.qrc) and QML type files (.qmltype) from a .pyproject file.

import resources_rc

Attenzione a non eliminare inavvertitamente la import resources_rc ad esempio mediante l'ottimizzatore di codice dell'IDE. Il presente listato non accede direttamente a metodi o variabili del modulo resources_rc, per cui un IDE potrebbe ritenere che la sua import non serva. Tuttavia l'eseguirne l'import Ã¨ sufficiente a caricare le definizioni delle risorse in esso contenute, al termine di resources_rc.py c'e': qInitResources() che carica nel sistema Qt le risorse in esso definite.

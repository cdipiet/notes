  • https://pypi.org/project/keyrings.cryptfile/ con la variabile di ambiente KEYRING_CRYPTFILE_PASSWORD si evita la richiesta di pwd interattiva

    from getpass import getpass
from os import getenv

keyring.set_keyring(kr)

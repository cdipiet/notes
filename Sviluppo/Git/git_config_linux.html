  5. aggiunta della chiave pubblica SSH all'account github:

    Per poter accedere a GitHub tramite SSH bisogna generare la coppia di chiavi privata/pubblica e configurare in GitHub la chiave pubblica.
    La directory .ssh puÃ² essere copiata in altri PC in modo da riutilizzare la chiave SSH per accedere a GitHub,
    ovvero si utilizza un unica chiave SSH per accedere a GitHub da diversi PC. In alternativa su ogni PC viene generata una coppia di chiavi privata/pubblica e si configura in GitHub la chiave pubblica di ogni PC.

    xclip -sel clip < ~/.ssh/id_rsa.pub # Copies the contents of the id_rsa.pub file to your clipboard

    In github, dall'account andare in https://github.com/settings/keys /Settings/SSH and GPG keys ed aggiungere la chiave SSH pubblica presente nella clipboard

  6. Test della connessione SSH a github

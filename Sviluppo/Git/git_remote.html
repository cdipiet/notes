Cambiare la url del remote ðŸ”—

  1. Go to the remote host (for example, https://github.com/User/project).
  2. Follow the host's instructions to rename the project (will differ from host to host, but usually Settings is a good starting point).
  3. Go to your local repository directory (i.e., open a command prompt and change to the repository's directory). Determine the new URL (for example, git@github.com:User/project-new.git)
  4. Set the new URL using Git:git remote set-url origin git@github.com:User/project-new.git
$ git remote -v # (per verificare che il nome del remote sia origin)
$ git remote set-url origin git@git.host.it:projects/myproject/myproject.git
$ git remote -v

# Altri comandi
$ git remote rename origin new-origin
$ git remote rm origin
$ git remote add origin git@git.host.it:projects/myproject/myproject.git

Uscire ðŸ”—

ZQ Esce senza salvare (same as :q!) :help ZQ

:q! chiude il buffer corrente :qa! chiude tutti i buffer aperti ed esce senza salvare

Load ðŸ”—

  • :e filename Open a new file. You can use the Tab key for automatic file name completion, just like at the shell command prompt.
  • :r file.txt read file after line
  • :23r infile Insert the content of infile under line 23
  • :r !program read program output
  • find . | vim - create a new file from standard input
  • :.!program replace line with program output

vi test.txt +4 Apre il file e si posiziona alla riga 4

Save ðŸ”—

  • :wq!
  • :wq
  • ZZ Write current file, if modified, and quit (same as :x) :help ZZ
  • :x!
  • :1,10 w outfile Saves lines 1 to 10 in outfile
  • :1,10 w >> outfile Appends lines 1 to 10 to outfile

Salvare parte di un file ðŸ”—

  • :.,$w nuovofile.txt persalvare il file dalla riga corrente alla fine in un nuovo file:
  • :w %.new il % indica il nome del file corrente
  • :n,m w! newfile salva le righe da n ad m nel nuovo file
  • :340,$w >> newfile
  • :w file write file (current file if no name given)
  • :w >> file append file (current file if no name given)

in modo Visual ðŸ”—

Selezionare in modo Visual v delle righe, e poi :w <nomefile> per salvare in un nuovo file la porzione di testo selezionata.

Revert ðŸ”—

:e!

is also useful. It discards your edits and returns to the last saved version of the current file.

Creare Un nuovo File ðŸ”—

  • :e newfile

  • :w nomefile salva il buffer corrente con il nome di file indicato

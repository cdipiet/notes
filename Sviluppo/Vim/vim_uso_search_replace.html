Per effettuare la ricerca ignore case:

:h ic
:set ic

For basic searching:

  • /pattern - search forward for pattern
  • ?pattern - search backward
  • n - repeat forward search
  • N - repeat backward

Some variables you might want to set:

  • :set ignorecase - case insensitive
  • :set smartcase - use case if any caps used
  • :set incsearch - show match as search proceeds
  • :set hlsearch - search highlighting

More cool searching tricks:

  • * - search for word currently under cursor
  • g* - search for partial word under cursor (repeat with n)
  • ctrl-o, ctrl-i - go through jump locations
  • [I - show lines with matching word under cursor

Replace ðŸ”—

:h range
:h substitute

:range s[ubstitute]/pattern/string/cgiI

Quando il range viene omesso la sostituzione avviene nella riga corrente. Per effettuare la ricerca in tutto il file utilizzare % come range e l'opzione global g.

For each line in the range replace a match of the pattern with the string where:

  • c Confirm each substitution
  • g Replace all occurrences in the line (without g - only first).
  • i Ignore case for the pattern.
  • I Don't ignore case for the pattern.

Cerca e sostituisci con conferma da commandline (:) ðŸ”—

:%s/prima/dopo/gc

Suppose you want to replace every occurrence of the word "idiot" with the word "manager".
But you want the chance to review each change before you do it.

Execute the command:

:%s/\<idiot\>/manager/gc

togliendo la c finale viene effettuato il search/replace immediatamente su tutto il file senza conferma

This command will make the change and pause after each change to give you a chance to confirm it. You can enter y to accept the change or n to not accept it.

The parts of this command are:

Parte Descrizione
: Enter command mode
% Perform this command on all lines (% is a synomim for the first to last line.)
s The short form of the :substitute command.
/\<idiot\>/ This text specifies the text we are looking for wand want. The \< tells Vim to match a word start and the \> tells Vim to match the end of a word.
/manager/ The replacement text
gc The flags. These are
g Global -- Change every occurance, not use the first one on each line
c Confirm -- Ask before making each change

Cerca e sostituisci ottenuto con comandi ðŸ”—

  • change - si usa con un comando di movimento per indicare il range di modifica
  • ce cambia dalla posizione attuale alla fine della parole corrente
  • cw cambia dalla posizione attuale all'inizio della parola seguente
  • c$ cambia dalla posizione del cursore al termine della riga
  • c0 cambia dall'inizio della riga alla posizione del cursore
  1. 1G Go to the top of the document
  2. /idiot<enter> Find the first occurrence of the word "idiot"
  3. cwmanager Change the word (cw) to manager.
  4. n Repeat the last search (find the next idiot.)
  5. . Repeat the last edit (change one word to manager) (If you do not want to change the word, skip this step.)

Repeat steps 4 and 5 until you have replaced all occurred.

Ricerche globali ðŸ”—

Use :g! (or its synonym :v) to search for all lines that do not contain pattern.

:g/pattern/

Finds (moves to) the last occurrence of pattern in the file.

:g/pattern/p

Finds and displays all lines in the file containing pattern.

:g!/pattern/nu

Finds and displays all lines in the file that don't contain pattern; also displays line number for each line found.

:60,124g/pattern/p

Finds and displays any lines between 60 and 124 containing pattern.

g can also be used for global replacements.

For example, to search for all lines that begin with WARNING: and change the first word not on those lines to NOT:

:g/^WARNING:/s/\<not\>/NOT/

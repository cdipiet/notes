Creation of symbolic links requires the SeCreateSymbolicLinkPrivilege (‚ÄúCreate symbolic links‚Äù) üîó

which is granted only to administrators by default (but you can change that using security policy).

To change the policies:

  1. Right click on Start‚ÜíRun and launch secpol.msc.
  2. Open Security Settings ‚Üí Local Policies ‚Üí User Rights Assignment and select Create symbolic links (that represents SeCreateSymbolicLinkPrivilege).
  3. Double-click on the item and add yourself (or the whole Users group) to the list.

The changes will apply when you log out and log in again.

Il pacchetto AUR per funzionare ha bisogno del pacchetto .deb scaricato da netacad

$ mkdir -p ~/app/cisco_packet_tracer/
$ cd ~/app/cisco_packet_tracer/
$ git clone https://aur.archlinux.org/packettracer.git
# se si Ã¨ scaricato l'archivio invece di utilizzare git: `tar -zxf packettracer.tar.gz`

  • A questo punto abbiamo il file: packettracer/PKGBUILD
  • Log into Cisco Networking Academy using your Cisco Account.
  • Register an account if you do not have it.
  • Once in the Networking Academy portal, download the Packet Tracer from Resources menu > Download Packet Tracer. The downloaded filename should be something like CiscoPacketTracer_801_Ubuntu_64bit.deb
  • Place the .deb file inside the folder where packettracer's build files are locatedâ€”that is the same folder where the PKGBUILD file is found.
$ mv ~/Scaricati/CiscoPacketTracer_801_Ubuntu_64bit.deb ~/app/cisco_packet_tracer/packettracer/
$ makepkg -s         # (it creates a packettracer.pkg.tar.xz)
$ makepkg -i         # or sudo pacman -U packettracer.pkg.tar.xz

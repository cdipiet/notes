Nomi delle interfacce di rete con systemd ðŸ”—

  • https://systemd.io/PREDICTABLE_INTERFACE_NAMES/ Starting with v197 systemd/udev will automatically assign predictable, stable network interface names for all local Ethernet, WLAN and WWAN interfaces.
    Assigning fixed names based on firmware/topology/location information has the big advantage that the names are fully automatic, fully predictable, that they stay fixed even if hardware is added or removed.
    Names incorporating physical/geographical location of the connector of the hardware (example: enp2s0).

IP Statico Utilizzando systemd ðŸ”—

Bisogna disabilitare NetworkManager ed abilitare il servizio di rete systemd.

$ systemctl --state=help
$ systemctl list-unit-files --no-pager '*[Nn]et*'
UNIT FILE                            STATE    VENDOR PRESET
netctl.service                       disabled disabled     
NetworkManager.service               enabled  disabled     
systemd-networkd.service             disabled enabled      
network.target                       static   -     

$ systemctl status systemd-networkd
$ systemctl cat systemd-networkd
$ man systemd-networkd.service
$ man systemd.network

$ sudo systemctl disable --now NetworkManager.service
$ sudo bash -c "cat > /etc/systemd/network/enp5s0.network" << 'EOF'
[Match]
Name=enp5s0

[Network]
Address=192.168.1.10/24
Gateway=192.168.1.1
DNS=8.8.8.8
DNS=8.8.4.4
EOF
$ sudo systemctl enable --now systemd-networkd.service

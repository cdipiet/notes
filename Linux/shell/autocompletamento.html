Autocompletamento ðŸ”—

  • il comando compgen genera delle liste di completamento, ad esempio compgen -d genera un lista delle sottodirectory della directory corrente
  • le regole di autocompletamento, dette compspec, sono aggiunte alla shell con il comando interno complete
  • all'avvio Bash, legge il file /etc/bash_completion per caricare le comspec
  • conviene inserire separate eventuali nuove funzioni e regole di autocompletamento in /etc/bash_completion.d

Mettere nella variabile CDPATH le directory maggiormente utilizzate, es.:

export CDPATH=.:/home/cdipiet/:/home/cdipiet/cdp:/media/Software

Supponiamo di avere installato il file /etc/bash_completion e di richiamarlo dal /etc/profile o .bashrc

Aggiunta del completamento mediante ricerca LDAP al comando ssh ðŸ”—

Aggiungere al file /etc/bash_completion :

_ldapcomplete(){
COMPREPLY=()
cur=${COMP_WORDS[COMP_CWORD]}
output=`ldapsearch -Q "(&(cn=$cur)(objectClass=ipHost))" cn | grep ^cn:`
name=${output#}
COMPREPLY=( ${COMPREPLY[@]} $name )
return 0
}
complete -F _ldapcomplete ssh

in $cur c'e' cio che si sta cercando di completare

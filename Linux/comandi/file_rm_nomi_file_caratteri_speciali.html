Se vogliamo cancellare un file il cui nome inizia con il carattere meno - non possiamo usare semplicemente il comando rm, questo perchÃ¨ rm interpreterebbe il - come l'inizio di una sua opzione. Per impedire ad rm di interpretare il - all'inizio del nome del file, usiamo il doppio meno --, che indica a qualsiasi comando di shell che l'elenco delle opzioni Ã¨ terminato e non ce ne saranno altre.

The double-dash -- prefixes long (verbatim) options to commands.

sort --ignore-leading-blanks

Used with a Bash builtin, it means the end of options to that particular command.

Tip

This provides a handy means of removing files whose names begin with a dash.

# Esempio 1:
bash$ ls -l  
\-rw-r--r-- 1 bozo bozo 0 Nov 25 12:29 -badname  
  
bash$ rm -- -badname  
  
bash$ ls -l  
total 0  
  
# Esempio2:  
for i in ./*[[:upper:]]*; do mv -- "$i" "${i,,}";done

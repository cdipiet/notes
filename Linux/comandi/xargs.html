Come xargs consuma gli argomenti ðŸ”—

xargs richiama il comando con il maggior numero di argomenti possibile per volta, numero di argomenti massimo dipendente da dei limiti fissati:

https://www.gnu.org/software/findutils/manual/html_node/find_html/Limiting-Command-Size.html

xargs gives you control over how many arguments it passes to the command each time it executes it. By default, it uses up to ARG_MAX - 2k, or 128k, whichever is smaller, characters per command. It uses as many lines and arguments as fit within that limit.

Spostare i soli file che non contengono una stringa ðŸ”—

Vogliamo spostare i soli file che non contengono require('expect'):

grep -LZ "require('expect')" * | xargs -I{} -0 mv {} ../../../hello_world/

grep -L per ottenere in output i nomi dei file che non corrispondono alla stringa di ricerca
grep -Z per utilizzare \0 come separatore dei nomi dei file prodotti in output, utile per gestire file i cui nomi possono contenere \n o spazi
xargs -0 per ricevere una lista di elementi separati dal carattere \0

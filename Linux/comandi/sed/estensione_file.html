#!/usr/bin/env bash

# Removes the suffix (from the last dot until the end of the string)
# from each of the strings passed, one per line, via STDIN
# Example of use:
# echo "foo.bar.txt" | remove_suffix
# will output "foo.bar" (without the quotes)

sed 's/\.[^\.]*$//'

s substitute whatever matches the pattern inside the first pair of / with the string supplied inside the second pair of /

/ start of pattern

\. a dot

[^.] any character that is not a dot

* match an arbitrary number of the preceding

$ the end of the line

/ end of pattern

nothing (what is to be substituted for the characters that match the above pattern)

/ end of substitution

ls -1 | sed 's/\(.*\)\..*/\1/'

${parameter%word} utile per rimuovere parti di stringa in coda The word is expanded to produce a pattern just as in pathname expansion. If the pattern matches a trailing portion of the expanded value of parameter, then the result of the expansion is the expanded value of parameter with the shortest matching pattern (the % case) or the longest matching pattern (the %% case) deleted.

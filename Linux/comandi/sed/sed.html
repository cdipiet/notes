    https://stackoverflow.com/questions/24889346/how-to-uncomment-a-line-that-contains-a-specific-string-using-sed

    ```bash
# To comment line containing specific string with sed, simply do:
$ sed -i '/<pattern>/s/^/#/g' file
# And to uncomment it:
$ sed -i '/<pattern>/s/^#//g' file
# Esempio:
$ sed -i '/2001/s/^/#/g' file    # (to comment out)
$ sed -i '/2001/s/^#//g' file    # (to uncomment)
```

File immutabili non cancellabili ðŸ”—

chattr +i myfile.txt

as root will make that file virtually indestructable no matter what user you are. If you want to remove the shield of steel, use -i rather than +i.

chattr -i myfile.txt
rm myfile.txt

Con chattr +i nemmeno root puo cancellare il file

Disk usage of all subdirectories and files including hidden files within the current directory (sorted by filesize) ðŸ”—

$ du -sk .[!.]* *|sort -n

    du -sk * | sort -nr | awk '
    BEGIN {
    split("KB,MB,GB,TB", Units, ",");
    }
    {
    u = 1;
    while ($1 >= 1024) {
    $1 = $1 / 1024;
    u += 1
    }
    $1 = sprintf("%.1f %s", $1, Units[u]);
    print $0;
    }
    '
    cd /
    du -ckx |sort -n
    cd ~
    du -ck |sort -n

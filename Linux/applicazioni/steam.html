    If you have a 64-bit system, enable the multilib repository:

    To use the multilib repository, uncomment the [multilib] section in /etc/pacman.conf(Please be sure to uncomment both lines):

    [multilib]
Include = /etc/pacman.d/mirrorlist

    Then upgrade the system and install the desired multilib packages.

    Tip: Run pacman -Sl multilib to list all packages in the multilib repository. 32-bit library package names begin with lib32-. pacman -Syu

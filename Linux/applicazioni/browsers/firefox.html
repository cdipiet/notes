Associare correttamente i file .maff a Firefox 🔗

Ho salvato molta documentazione da Firefox in formato .maff(Mozilla Archive File Format), tuttavia siccome è associato al tipo di file archivio, facendo doppio click da Nautilus in GNOME, se si associa il .maff a Firefox, allora anche i file .ziplo saranno, analogamente riassociando un .zip a “File Roller”, anche i .maff saranno associati al gestore di file compressi... Per risolvere il problema bisogna definire un nuovo tipo mime per i file .maff ed associarlo a Firefox.

  1. Creare il file:

    ~/.local/share/mime/packages/maff.xml

    Con il contenuto:

    <?xml version="1.0" encoding="UTF-8"?>
<mime-info xmlns="http://www.freedesktop.org/standards/shared-mime-info">
<mime-type type="application/x-maff">
<comment>Mozilla Archive Format</comment>
<glob pattern="*.maff"/>
</mime-type>
</mime-info>

  2. Aggiornamento DB MIME

    sudo update-mime-database ~/.local/share/mime

3- Doppio click su un file .maff e selezionare Firefox come applicazione associata

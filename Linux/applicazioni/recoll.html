Recoll File Indexer 🔗

Installazione 🔗

https://www.lesbonscomptes.com/recoll/

Indice:

# Per cancellare l'indice
rm -rf ~/.recoll/xapiandb

Versione dei repository Ubuntu 🔗

sudo apt install recoll aspell-it
sudo pacman -Syu --needed --noconfirm recoll aspell-it

Versione più aggiornata in Mint 🔗

https://launchpad.net/~recoll-backports/+archive/ubuntu/recoll-1.15-on

sudo apt install aspell-it
sudo add-apt-repository ppa:recoll-backports/recoll-1.15-on
sudo apt-get update
sudo apt install recoll

Lanciare Recoll con l'interfaccia in Inglese 🔗

La traduzione italiana è incompleta con il risultato di avere i menu dell'applicazione in parte in italiano ed in parte in inglese, meglio avere tutto in inglese.

Essendo una applicazione Qt5 basta configurare il locale prima di lanciarla:

env LANGUAGE=en recoll &

In Mint utilizzare cinnamon-menu-editor che equivale a:

cp /usr/share/applications/recoll-searchgui.desktop ~/.local/share/applications
vim ~/.local/share/applications/recoll-searchgui.desktop

oppure

In sudo vim /usr/share/applications/recoll-searchgui.desktop aggiungere LANGUAGE=en

$ cat /usr/share/applications/recoll-searchgui.desktop
[Desktop Entry]
Categories=Qt;Utility;Filesystem;Database;
Comment=Find documents by specifying search terms
Comment[ru]=Поиск документов по заданным условиям
Exec=env LANGUAGE=en recoll
GenericName=Local Text Search
GenericName[ru]=Локальный текстовый поиск
Icon=recoll
Name=Recoll
Terminal=false
Type=Application
Keywords=Search;Full Text;

Configurazione 🔗

Preferenze/Index Configuration

    Parametri Globali
        Nome della cartella del database: xapiandb
        Lingue per la radice: english, italian
        Indirizzi saltati: /media
    Parametri Locali/Globale/Nomi saltati
        node_modules

Indicizzazione continua 🔗

~/.config/autostart/recollindex.desktop

[Desktop Entry]
Name=Recoll real time indexer
Comment=Runs in background to extract and index text from modified documents
Icon=system-run
Exec=recollindex -w 60 -m
Terminal=false
TerminalOptions=
Type=Application
Categories=Utility;Filesystem;Database;
NoDisplay=true
X-GNOME-Autostart-enabled=true
X-KDE-autostart-after=panel
X-KDE-UniqueApplet=true

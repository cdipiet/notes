Exec variables ðŸ”—

Add Accepts
%f a single filename.
%F multiple filenames.
%u a single URL.
%U multiple URLs.
%d a single directory. Used in conjunction with %f to locate a file.
%D multiple directories. Used in conjunction with %F to locate files.
%n a single filename without a path.
%N multiple filenames without paths.
%k a URI or local filename of the location of the desktop file.

Creare Icone per nuove Applicazioni ðŸ”—

Si puÃ² utilizzare l'applicazione MenuLibre:

pacman -S menulibre

Oppure Manualmente si crea un file .desktop:

  • https://wiki.archlinux.org/index.php/desktop_entries
  • https://developer.gnome.org/integration-guide/stable/desktop-files.html.en

Place this file in the /usr/share/applications directory so that it is accessible by everyone,
or in ~/.local/share/applications if you only wish to make it accessible to a single user.

(~/.config/autostart/ per l'autopartenza all'avvio della sessione gnome)

  • https://standards.freedesktop.org/
  • https://standards.freedesktop.org/autostart-spec/
  • https://standards.freedesktop.org/desktop-entry-spec/

Esempio:

$ cat <<'EOF' > ~/.local/share/applications/jetbrains-pycharm-ce.desktop
[Desktop Entry]
Version=1.0
Type=Application
Name=PyCharm Community Edition
Comment=Python IDE
TryExec=/home/$USER/app/pycharm-community-2021.1.1/bin/pycharm.sh
Exec="/home/$USER/app/pycharm-community-2021.1.1/bin/pycharm.sh" %f
Icon=/home/$USER/app/pycharm-community-2021.1.1/bin/pycharm.svg
Keywords=IDE;Python;PyCharm;
Encoding=UTF-8
NoDisplay=false
Categories=Development;IDE;TextEditor;
StartupWMClass=jetbrains-pycharm-ce
StartupNotify=false
Terminal=false
EOF

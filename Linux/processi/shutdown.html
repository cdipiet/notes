Kernel (SysRq) ðŸ”—

Ci sono diverse combinazioni di tastiera che possono inviare segnali SysRq al Kernel mediante la combinazione di tasti ALT+SysRq. In alcune tastiere il tasto SysRq si chiama Print Screen o Stamp.

  • https://wiki.archlinux.org/title/Keyboard_shortcuts#Kernel_(SysRq)
  • https://wiki.archlinux.org/title/Sysctl
  • https://www.kernel.org/doc/html/latest/admin-guide/sysrq.html
  • https://github.com/systemd/systemd/blob/main/sysctl.d/50-default.conf#L14-L19
  • https://en.wikipedia.org/wiki/Magic_SysRq_key

Di default systemd imposta i permessi sysrq a 0x10,

https://github.com/systemd/systemd/blob/main/sysctl.d/50-default.conf#L14-L19

# System Request functionality of the kernel (SYNC)
#
# Use kernel.sysrq = 1 to allow all keys.
# See https://www.kernel.org/doc/html/latest/admin-guide/sysrq.html for a list
# of values and keys.
kernel.sysrq = 16

Il valore 0x10 come si vede da https://www.kernel.org/doc/html/latest/admin-guide/sysrq.html#how-do-i-enable-the-magic-sysrq-key:

1  = enable all functions of sysrq
16 =  0x10 - enable sync command

abilita solamente l'invio del comando sync, il che non consente di inviare il segnale di shutdown o reboot. Bisogna impostare kernel.sysrq = 1 nella configurazione sysctl per abilitare tutti i comandi sysrq inviabili da tastiera al kernel.

  • https://www.kernel.org/doc/html/latest/admin-guide/sysrq.html#how-do-i-enable-the-magic-sysrq-key

I comandi:

sysctl kernel.sysrq=1 o echo "1" > /proc/sys/kernel/sysrq

non cambiano in maniera persistente l'impostazione del parametro, sono temporanei ed al reboot vengono perduti.

Abilitazione della combinazione tasti ALTGr+Rsist ðŸ”—

Verifica abilitazione kernel

$ grep CONFIG_MAGIC_SYSRQ /boot/config*
CONFIG_MAGIC_SYSRQ=y

Abilitazione runtime

il parametro del kernel /proc/sys/kernel/sysrq deve valere 1:

$ cat /proc/sys/kernel/sysrq
$ echo 1 > /proc/sys/kernel/sysrq

oppure:

$ sysctl -w kernel.sysrq=1

Per rendere l'abilitazione permanente inserire la seguente riga nel file /etc/sysctl.d/99-sysctl.conf:

kernel.sysrq=1

Modo consigliato di riavvio del computer quando e' bloccato ðŸ”—

R>E>I>S>U>B

Radunare Elefanti Indiani Su Una Barca

AltGr+Rsyst+...: REISUB

Tasto Funzione
0-9 imposta il livello di log di comandi come M e P
R passa la tastiera in modalita XLATE, riprende il controllo della tastiera
E invia il segnale SIGTERM a tutti i processi tranne init, modo pulito di chiudere le applicazioni
I invia il segnale SIGKILL a tutti i processi escluso init. Terminazione forzata
S scrive su disco tutti i dati contenuti nei buffer
U rimonta tutti i filesystem in sola lettura
B esegue un riavvio immediato
O spegne il computer
M mostra sulla console dati statistici sull'uso della memoria
T mostra sulla console la lista dei task
H mostra un breve sommario di tutti i tasti R syst

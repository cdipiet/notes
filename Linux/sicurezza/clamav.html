Aggiornamento Database ðŸ”—

Per vedere lo stato di aggiornamento:

$ clamscan --version
ClamAV 0.103.2/26204/Thu Jun 17 13:19:00 2021
$ cat /var/log/clamav/freshclam.log

Per aggiornare il database bisogna che il demone sia avviato:

$ sudo systemctl start clamav-daemon.service`

Dopo aggiorna la definizione dei virus con:

$ sudo freshclam

I file del database sono salvati in:

/var/lib/clamav/daily.cvd
/var/lib/clamav/main.cvd

Si vede con:

$ grep var /etc/clamav/clamd.conf 
LogFile /var/log/clamav/clamd.log
# copy the configuration file, change the LogFile variable, and run
# Default: system specific (usually /tmp or /var/tmp).
# DatabaseDirectory /var/lib/clamav

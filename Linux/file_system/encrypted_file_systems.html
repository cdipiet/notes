LUKS, Linux Unified Key Setup, is a standard for hard disk encryption.

Cryptsetup is utility used to conveniently setup disk encryption based on dm-crypt kernel module. These include plain dm-crypt volumes, LUKS volumes, loop-AES and TrueCrypt compatible format.

Create encrypted partition

#dd if=/dev/urandom of=/dev/sdc1# Optional. For paranoids only (takes days)
cryptsetup -y luksFormat /dev/sdc1# This destroys any data on sdc1
cryptsetup luksOpen /dev/sdc1 my_crypto_disk
mkfs.ext3 /dev/mapper/my_crypto_disk# create ext3 file system
mkfs.vfat /dev/mapper/my_crypto_disk -n my_crypto_disk
mount -t ext3 /dev/mapper/my_crypto_disk /mnt
umount /mnt
cryptsetup luksClose my_crypto_disk# Detach the encrypted partition

# Attach

cryptsetup luksOpen /dev/sdc1 my_crypto_disk
mount -t ext3 /dev/mapper/my_crypto_disk /mnt

# Detach

umount /mnt
cryptsetup luksClose my_crypto_disk
